About

BluestNight is a powerful theme for Hugo written by Michael Bryant (aka Shadow53)

Supported browsers

The following are the earliest versions of each browser compatible with this theme

  • Chrome/Chromium 32.0
  • Firefox 29.0
  • Microsoft Edge
  • Safari 9.0
  • Opera 19

Documentation

You can find documentation on how to use the theme, as well as a config.toml template, in the documentation section.

See the theme in action with its default color scheme here or with an override here.

Feature overview

  • Responsive design
  • Customizable color scheme and background
  • Print-friendly CSS
  • Powerful custom shortcodes:
    • Netlify compatible forms
    • Responsive tables
    • Smart links to static files
    • Member boxes - like author boxes that can be put anywhere on a page
    • Royalty-free custom fonts
  • Site-wide and per-page alert boxes
  • Sidebar widgets, including:
    • Patreon banner linking to your Patreon page. Automatically detects white or black text based on page background color
    • Search powered by Lunr.js
    • List of upcoming events with a button to subscribe to the calendar (.ics file)
    • List of the five most popular categories
    • Tag cloud, sorted by popularity
    • Ten most recent posts with a link to subscribe via RSS
  • Comments powered by Hashover or Muut
  • Analytics powered by Piwik
  • Social icons linking to your social media pages and more. Supports 22 websites as of 09/14/2017
  • Syntax highlighting powered by Pygments
  • Custom index pages (_index.md) with custom content, sorting, and more
  • Custom robots.txt, 404 page, and RSS feed templates
  • And more!
